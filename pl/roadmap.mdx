## Wydane

### 2024

**Q1**
- Aktualizacja aplikacji AI — wsparcie dla Apple MacBook M2/M3 (później wycofane dla nie-NVIDIA po stronie AI)
- Web LLM dla społeczności GamerHash zaktualizowany do GPT-4 Turbo (wycofany po pojawieniu się modeli lokalnych)
- Airdrop GHX dla posiadaczy GHXP, aktualizacja aplikacji mobilnej

**Q2**
- Uruchomienie **GHXP Boost** — wyższe nagrody za GHX trzymany w MetaMask
- Nowy dashboard AI Chat, FAQ w EN
- Pierwsze wewnętrzne testy modułu Earn
- Strona statystyk z mapą urządzeń; landing page GHXP

**Q3**
- Domyślny model Chatu zmieniony na **Llamę 3.1**; poprawki walidacji GPU
- AI Image Booster — prompt enhancer bramkowany GHX
- Strona z brand i media kitem
- **Model Center** plus nowe modele: Llama 3.1/3.2, Qwen 2.5, Gemma 2, warianty Stable Diffusion, DreamShaper
- Płatności USDT w GamerHash Wallet

**Q4**
- Nowe modele do generowania obrazów: Sana 1.6B
- Pakiety rozszerzeń LoRA
- Pełny redesign [gamerhash.com](https://gamerhash.com) — desktop i mobile
- Dodano **Phi-4** (Microsoft); modele tekstowe i koderskie **DeepSeek**
- Odświeżona strona główna i nawigacja statystyk, tłumaczenia EN/PL/KO/PT

### 2025

**Q1**
- Duża paczka rozszerzeń LoRA w aplikacji AI
- Aktualizacja koparek: GMiner, Nanominer, XMRig, lolMiner (ostatnie aktualizacje po stronie miningu)
- Wsparcie dla **NVIDIA Blackwell RTX 5xxx** + AMD RX9070
- Generowanie obrazów na RTX 5xxx
- **Bridge GHX: Ethereum ↔ Solana** (Raydium, Jupiter, Solscan)
- Dodano **Qwen 3, Llama 3.2, Gemma 3, Phi 4, DeepSeek R1, Bielik**
- **HiDream, Sana Sprint** (obraz)
- Redesign onboardingu aplikacji, przebudowa backendu i UI

**Q2**
- Kolejne pakiety modeli LoRA
- Integracja z GenTok (boty Telegram / Discord)
- **GHX × Solana — staking i Liquidity Mining** na [staking.gamercoin.com](https://staking.gamercoin.com)
- Aplikacja AI 0.25.10 — **AI Image → Video** (img2vid, txt2vid), modele **LTX-Video**
- Aplikacja AI 0.26.0 — wsparcie dla **GPT-OSS-20B** (otwarte wydanie OpenAI)

**Q3**
- Nowe LoRA w kreatywnym stylu na FLUX Schnell
- Nowe przestrzenie społecznościowe GenTok AI
- Generatywne boty AI rozszerzone na Telegram + Discord
- **Publiczne API dla deweloperów** — beta release udostępniająca sieć GamerHash
- W aplikacji AI: image-to-video i text-to-video, booster prompts, nowa nawigacja, zaawansowane modele wideo
- Publiczne API dla deweloperów — aplikacja webowa i pełen otwarty dostęp dla wszystkich użytkowników
- Playground API dla deweloperów — text-to-image, text-to-speech, image-to-text, video-to-text
- Aplikacja mobilna — wsparcie iOS, stabilność, **wypłaty USDC**
- Odświeżenia stron gamerhash.com i gamercoin.com

**Q4**
- **Wypłaty USDC przez Binance** w module Earn
- **Embeddingi** w Model Center — BGE M3
- Rozszerzenie TTS — **Chatterbox**, **Kokoro**
- **Whisper Large V3** — transkrypcja głosu i audio (audio + wideo, pliki + na żywo)
- Modele wideo **LTX-2** — 19B Distilled

### 2026

**Q1**
- Mining krypto wygaszony **31 stycznia 2026** — platforma w pełni skupiona na AI
- Moduł **Image Edit** — Qwen-Image-Edit Plus
- Moduł **Text-to-Music** — ACE-Step 1.5 (Base / Turbo + warianty XL)
- **Video Upscale** — FlashVSR Tiny, RealESRGAN Video x2/x4
- **Video Replace / Animate** — Wan2.2-Animate 14B
- **OCR / ekstrakcja dokumentów** — Nanonets OCR S
- Nowe modele obrazowe — **FLUX.2 Klein 4B**, **Z-Image-Turbo**
- Warianty **Qwen3 TTS** — VoiceDesign, VoiceClone, CustomVoice
- Model wideo **LTX-2.3 22B Distilled**

## Jak ustalane są priorytety

Roadmapę napędzają trzy sygnały:

1. **Popyt deweloperów** — jakich modeli i funkcji chcą konsumenci API.
2. **Feedback kontrybutorów aplikacji AI** — bugi i prośby o nowe funkcje z kanałów wsparcia bezpośrednio zasilają kolejny kamień milowy.
3. **Ekosystem open source** — gdy pojawia się mocny nowy model, jego integracja w Model Center idzie w fast track.
