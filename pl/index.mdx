GamerHash zamienia bezczynne, konsumenckie GPU w produktywną moc obliczeniową dla AI. Nasza flagowa aplikacja [**GamerHash AI App**](https://gamerhash.com/en/app-download) pozwala każdemu, kto ma kartę klasy RTX, uruchamiać prywatne narzędzia AI — czat, obrazy, wideo i głos — oraz zarabiać za udostępnianie sieci wolnej mocy GPU. Deweloperzy mogą korzystać z tej samej sieci przez publiczne API zgodne z OpenAI.

## Jak działa ekosystem GamerHash

<Frame>
  <img src="/images/ecosystem-flow.png" alt="Przepływ ekosystemu GamerHash: kontrybutorzy uruchamiają aplikację AI i udostępniają wolną moc GPU; sieć DePIN ją agreguje; deweloperzy AI i integratorzy korzystają z niej przez API i bezpośrednie integracje; przychody wracają do kontrybutorów jako nagrody — GUSD za zadanie i GHX dystrybuowane w sezonach GHXP." />
</Frame>

Gracze udostępniają bezczynny czas GPU. Sieć DePIN agreguje go i kieruje zadania inferencji, generowania obrazu i wideo do dostępnych maszyn. Deweloperzy AI i integratorzy płacą za uzyskaną moc obliczeniową przez warstwę API lub bezpośrednie integracje. Przychody wracają do kontrybutorów jako nagrody — a pętla wzmacnia się, gdy dołączają kolejni partnerzy.

## Partnerzy i integracje

GamerHash i token GHX integrują się z dostawcami infrastruktury, zespołami AI i protokołami Web3, w tym:

[Aethir](https://aethir.com) · [BNB Chain](https://www.bnbchain.org) · [Celer Network](https://celer.network) · [Chainlink](https://chain.link) · [Exabits](https://exabits.ai) · [IQNet](https://iqnet.io) · [PAAL AI](https://paalai.ai) · [The Sandbox](https://www.sandbox.game)

Nasze publiczne API dla deweloperów dostępne jest jako oficjalny node [n8n](https://n8n.io), więc każda aplikacja lub workflow może kierować zadania AI do sieci bez budowania własnej infrastruktury.
