---
title: "AI Image Generation"
description: "Distributed AI image generation."
---

This module involves **uses computational power** of the user's **graphics card** , which is used to **generate images**. The user does not rent out any computational power or share it with anyone else. This module is exclusively for the user of the application and is fully **available for free**. This image generation module currently utilizes a **text-to-image model** , meaning that based on entered prompts, the user will be able to generate images according to his needs. In the future, it will also be possible to use an **image-to-image** and **image-to-video** model. It is important to note that **no** **prompts** or **images** are **stored by us** in the database.

This module provides many functionalities where the user can adjust the number of generated images, their quality, fidelity, and more. The application also allows storing already generated images as user history (locally on user’s computer) and provides a full-screen preview of the generated image. Users can input specific prompts but also exclude certain words using dedicated functionality.
