## 일반적인 셋업

1. **게이밍 PC에 [AI App](https://gamerhash.com/en/app-download)을 설치**합니다. 무료, Windows 전용.
2. **프라이빗 AI를 사용하고 싶을 때마다 실행**합니다 — Llama로 채팅, FLUX로 이미지 생성, LTX로 비디오 클립 실행. 로컬, 빠름, 사용량 제한 없음.
3. 잠자리에 들거나 출근하기 전에 **수익 모듈을 켭니다**. 게임과 충돌하지 않도록 GPU 사용량 상한을 설정하세요.
4. 아침에 [모바일 앱](/ko/apps/mobile)을 **확인**합니다 — PC가 밤새 얼마를 벌었는지 확인하세요.
5. [스토어](/ko/apps/store)에서 **GUSD를 사용**하거나 GHX로 전환하여 HODL하세요.

## 왜 유료 AI 서비스보다 나은가

- 클라우드 AI(ChatGPT Plus, Midjourney, Sora)는 월 $20–60의 비용이 들고 사용량을 제한합니다. 이미 소유한 동일한 GPU에서 실행되는 로컬 AI는 추가 비용이 없습니다.
- 게임에 사용하는 동일한 GPU가 **더 크고 검열이 적은 오픈 소스 모델**을 프라이빗하게 실행합니다.
- 게임을 하지 않는 시간에 보상을 받습니다.

## 엣지 케이스

- **토너먼트 / 경쟁 게이밍 셋업?** 게임 시간 동안 GPU 사용량 상한을 0%로 설정하세요. 수익 모듈이 VRAM을 두고 경쟁하지 않습니다.
- **오래된 GPU?** 8 GB로 채팅과 기본 이미지 생성을 잠금 해제할 수 있습니다. 가장 높은 수익의 워크로드는 받지 못하지만 개인 사용은 충분합니다.
- **여러 PC?** 각 PC에서 AI App을 실행하세요 — 수익은 하나의 계정에 통합되고, GHXP가 더 빠르게 누적됩니다.
- **암호화폐에 관심이 없다면?** GHX 측을 완전히 건너뛰세요. GUSD → 스토어 → 게임 키. 끝.
