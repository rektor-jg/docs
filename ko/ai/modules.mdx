AI App은 모듈형입니다. 전부 다 쓸 필요는 없습니다. 대시보드에서 필요한 것만 골라 쓰면 됩니다. 모든 모듈은 GPU에서 로컬로 실행됩니다.

## Earn 모듈

옵트인하면 PC가 유휴 상태일 때 GPU가 디스패치 큐에 합류합니다. AI 추론 워크로드가 머신으로 라우팅되며, 완료한 워크로드당 **GUSD**가 적립됩니다.

- **유휴 시간에 자동 수익화.** 워크로드는 GPU와 VRAM을 모두 사용하며, 게임을 실행하면 모듈이 자동으로 빠집니다. 게임 안 할 때 그냥 켜두면 됩니다.
- **언제든 일시 정지.** 최소 약정 없음 — 사이드바에서 토글로 끌 수 있습니다.

머신당 수익에 영향을 주는 요인은 [수익 모델](/ko/ai/earning) 문서에서 확인하세요.

## Home 모듈

백그라운드에서 동작하며 계정과 네트워크 보상을 연결하는 대시보드입니다.

- **GHXP 적립** — 앱이 켜져 있는 매 분마다 GHXP가 쌓입니다. 대기 중인 포인트는 매 정각에 정산됩니다. MetaMask 또는 인앱 지갑에 GHX를 보유하면 적립 속도가 배가됩니다.
- **시즌** — 두 달마다 누적된 GHXP를 락업해, 시즌 GHX 풀에서 비율만큼 보상을 받습니다.
- **계정 상태** — 현재 GHXP, 레벨, 시즌 순위, 연결된 지갑을 한 화면에서 확인합니다.

자세한 메커니즘은 [GHXP — 참여 배율](/ko/ai/ghxp) 문서를 참고하세요.

## 이미지

| 모듈 | 동작 | 모델 |
| --- | --- | --- |
| **Text-to-Image** | 프롬프트로 이미지 생성. 결과물은 로컬에 저장되며 앱이 디바이스별로 히스토리를 관리합니다. | Flux.1 dev, FLUX.1 Schnell 12B, FLUX.2 Klein 4B, Z-Image-Turbo, Sana Sprint 0.6B / 1.6B |
| **Image Edit** | 텍스트 지시로 기존 이미지 편집. 피사체 변경, 배경 교체, 스타일 변경, 인페인팅. | Qwen-Image-Edit Plus |
| **Image Upscale** | 재프롬프트 없이 기존 이미지의 해상도 업스케일. | RealESRGAN x2 / x4 |
| **Background Removal** | 피사체 누끼 따기. 합성이나 투명 에셋을 위한 깔끔한 알파 출력. | Ben2 |

**AI Image Booster** — Text-to-Image의 선택형 GHX 게이팅 기능. 보조 모델이 프롬프트 품질을 한 단계 끌어올려 줍니다.

## 비디오

| 모듈 | 동작 | 모델 |
| --- | --- | --- |
| **Image / Text / Audio → Video** | 정지 이미지, 텍스트 프롬프트, 오디오 트랙을 짧은 비디오 클립으로 변환. | LTX-2.3 22B Distilled, LTX-2 19B Distilled, LTX-Video 0.9.8 13B |
| **Video Upscale** | 기존 비디오 클립의 해상도 업스케일. | FlashVSR Tiny, RealESRGAN Video x2 / x4 |
| **Video Replace / Animate** | 기존 비디오의 피사체를 애니메이션화하거나 교체. 캐릭터 교체, 레퍼런스 모션 적용, 액션 스타일 변경. | Wan2.2-Animate 14B |

## 음성 및 오디오

| 모듈 | 동작 | 모델 |
| --- | --- | --- |
| **Text-to-Speech** | 텍스트로부터 음성 생성. 커스텀 보이스 디자인, 기존 음성 복제, 기본 보이스 사용. | Qwen3 TTS (VoiceDesign / VoiceClone / CustomVoice), Chatterbox, Kokoro |
| **Voice & Audio Transcription** | 오디오·비디오의 음성을 텍스트로 변환. 파일 또는 실시간 스트림 모두 지원. | Whisper Large V3 |
| **Text-to-Music** | 텍스트 프롬프트로 음악 생성. 게임 사운드트랙, 앰비언트 배경, 프로토타이핑에 활용. | ACE-Step 1.5 (Base / Turbo + XL 변형) |

## 텍스트 및 문서

| 모듈 | 동작 | 모델 |
| --- | --- | --- |
| **AI Chat** | 최신 오픈소스 LLM과 프라이빗 채팅. 사용량 제한 없음, 외부 서버로 아무것도 전송되지 않음. | Model Center에서 선택 |
| **OCR / Document Extraction** | 이미지와 스캔 문서에서 텍스트 추출. 영수증, 스크린샷, 페이지 사진 등. | Nanonets OCR S |
| **Embeddings** | 텍스트에서 벡터 임베딩 생성. 검색, RAG 파이프라인, 로컬 시맨틱 검색에 활용. | BGE M3 |

## Model Center

앱 내부의 모델 탐색·다운로드·전환 허브입니다. 본인 하드웨어 기준의 VRAM 적합도를 함께 표시합니다. 오픈소스 생태계에서 새 모델이 나올 때마다 여기에 정기적으로 추가됩니다.
