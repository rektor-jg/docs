**GamerHash Mobile App**은 **iOS**와 **Android**용 무료 컴패니언 앱입니다. 모바일 자체에서 AI 워크로드를 돌리지는 *않습니다*. Mobile은 GamerHash 계정의 대시보드이자 지갑 역할로, PC 앞에 앉아 있지 않아도 필요한 작업을 처리할 수 있도록 해줍니다.

## 주요 기능

  <Card title="계정 관리" icon="user-gear">
    GamerHash 계정 프로필 설정과 비밀번호 재설정.
  </Card>
  <Card title="실시간 수익 대시보드" icon="chart-line">
    적립된 GUSD, GHXP 잔고, 디바이스별 지급 이력.
  </Card>
  <Card title="워커 모니터링" icon="server">

  </Card>
  <Card title="지갑 전송" icon="wallet">
    GHX 또는 USDC를 외부 지갑으로 출금.
  </Card>
  <Card title="QR 코드 로그인" icon="qrcode">
    데스크톱 앱에 표시된 QR 코드를 찍어 빠르게 로그인. 같은 계정을 사용하므로 별도 인증 정보가 필요 없습니다.
  </Card>
</CardGroup>
