## 출시 완료

### 2024년

**1분기**
- AI App 업데이트 — Apple Macbook M2/M3 지원 (이후 AI 측에서 비-NVIDIA 지원 종료)
- AstroGPT가 GPT-4 Turbo로 업그레이드 (로컬 모델 도입과 함께 사용 중단)
- GHXP 보유자를 위한 GHX 에어드롭, 모바일 앱 업데이트

**2분기**
- **GHXP Boost** 출시 — MetaMask에 보유한 GHX에 대한 더 높은 보상
- 새 AI Chat 대시보드, 영어 FAQ
- 수익 모듈 첫 내부 테스트
- 디바이스 맵이 있는 통계 페이지, GHXP 랜딩 페이지

**3분기**
- 기본 채팅 모델이 **Llama 3.1**로 전환, GPU 검증 수정
- AI Image Booster 프롬프트 (100 GHX 유료 기능)
- 브랜드 및 미디어 키트 페이지
- **Model Center** 및 신규 모델: Llama 3.1/3.2, Qwen 2.5, Gemma 2, Stable Diffusion 변형, DreamShaper
- GamerHash Wallet의 USDT 결제

**4분기**
- 신규 이미지 생성 모델: Sana 1.6B, Astro GHX Style LoRA
- LoRA 확장 팩
- [gamerhash.com](https://gamerhash.com)의 전면 리디자인 — 데스크톱 및 모바일
- **Phi-4** (Microsoft) 추가, **DeepSeek** 텍스트 + 코더 모델
- 홈페이지 및 통계 내비게이션 리프레시, EN/PL/KO/PT 번역

### 2025년

**1분기**
- AI App에 대규모 LoRA 확장 드롭
- Miner 업데이트: GMiner, Nanominer, XMRig, lolMiner (마지막 마이닝 측 업데이트)
- **NVIDIA Blackwell RTX 5xxx** + AMD RX9070 지원
- RTX 5xxx에서 이미지 생성
- **GHX 브리지: 이더리움 ↔ Solana** (Raydium, Jupiter, Solscan)
- **Qwen 3, Llama 3.2, Gemma 3, Phi 4, DeepSeek R1, Bielik** 추가
- **HiDream, Sana Sprint** (이미지)
- 앱 온보딩 리디자인, 백엔드 + UI 개편

**2분기**
- 추가 LoRA 모델 팩
- GenTok 통합 (Telegram / Discord 봇)
- **GHX × Solana 스테이킹 & 유동성 마이닝** [staking.gamercoin.com](https://staking.gamercoin.com)
- AI App 0.25.10 — **AI Image → Video** (img2vid, txt2vid), **LTX-Video** 모델
- AI App 0.26.0 — **GPT-OSS-20B** (OpenAI의 오픈 릴리스) 지원

**3분기**
- FLUX Schnell의 새로운 크리에이티브 스타일 LoRA
- GenTok AI 새 커뮤니티 공간
- 생성 AI 봇이 Telegram + Discord로 확장
- **deAPI 공개 베타** — GamerHash 네트워크용 공개 API
- AI App image-to-video & text-to-video, 부스터 프롬프트, 새 내비게이션, 고급 비디오 모델
- **deAPI** 웹 앱 및 전체 사용자에 대한 공개 API 액세스
- deAPI 플레이그라운드 — text-to-image, text-to-speech, image-to-text, video-to-text
- 모바일 앱 — iOS 지원, 안정성, **USDC 지급**
- gamerhash.com 및 gamercoin.com 웹사이트 리프레시

## 우선순위는 어떻게 정해지는가

로드맵은 세 가지 신호에 의해 추진됩니다.

1. **deAPI 수요** — 개발자가 어떤 모델과 기능을 요청하는지.
2. **AI App 기여자 피드백** — 지원 채널과 앱 내 설문 조사가 다음 마일스톤에 직접 반영됩니다.
3. **오픈 소스 생태계** — 강력한 새 모델이 출시되면 Model Center에 통합하는 것을 빠르게 추진합니다.

이 로드맵은 사용자가 영향을 미칠 수 있습니다. 제품 피드백을 위한 가장 빠른 채널은 다음과 같습니다.
