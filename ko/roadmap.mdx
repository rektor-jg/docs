**Q3**

- 신규 이미지 생성 모델: Sana 1.6B

- [gamerhash.com](https://gamerhash.com) 전면 리뉴얼 — 데스크톱·모바일
- **Phi-4**(Microsoft) 추가, **DeepSeek** 텍스트·코더 모델

## 우선순위 결정 방식

로드맵은 세 가지 시그널을 보고 결정됩니다.

1. **개발자 수요** — API 사용자가 요청하는 모델과 기능.
2. **AI App 기여자 피드백** — 지원 채널의 버그 리포트와 기능 요청은 다음 마일스톤에 그대로 반영됩니다.
3. **오픈소스 생태계** — 새로 좋은 모델이 나오면 Model Center 통합을 빠르게 진행합니다.
