GamerHash는 유휴 상태인 일반 소비자용 GPU를 생산적인 AI 컴퓨팅 자원으로 전환합니다. 대표 제품인 [**GamerHash AI App**](https://gamerhash.com/en/app-download)을 사용하면 RTX급 GPU를 보유한 누구나 프라이빗 AI 도구(채팅, 이미지, 비디오, 음성)를 실행하고 남는 GPU 시간을 네트워크에 공유하여 보상을 얻을 수 있습니다. 동일한 네트워크는 개발자에게 [**deAPI**](https://deapi.ai) — OpenAI 호환 공개 API로 제공됩니다.

## 빠른 사실

| | |
| --- | --- |
| **설립** | 2017년 — 폴란드 |
| **사용자** | 전 세계 80만 명 이상 가입 |
| **대표 제품** | GamerHash AI App ([다운로드](https://gamerhash.com/en/app-download)) |
| **개발자 API** | [deAPI](https://deapi.ai) — 6,000명 이상의 개발자 등록 |
| **토큰** | [GamerCoin (GHX)](https://docs.gamercoin.com) — <Eth /> · <Bsc /> · <Sol /> |
| **마이닝** | 2026년 1월 31일 종료 — 플랫폼은 이제 AI 컴퓨팅으로 운영됩니다 |

## GamerHash 생태계의 작동 방식

<Frame>
  <img src="/images/ecosystem-flow.svg" alt="GamerHash 생태계 흐름: 기여자가 AI App을 실행해 유휴 GPU 자원을 공유하고, DePIN 네트워크가 이를 집계하며, AI 빌더와 개발자가 deAPI를 통해 사용하고, 수익이 사용자 보상과 GHX 매입·소각으로 환원됩니다." />
</Frame>

게이머는 유휴 GPU 시간을 기여합니다. DePIN 네트워크가 이를 집계하여 추론, 이미지, 비디오 워크로드를 사용 가능한 머신으로 라우팅합니다. AI 빌더와 개발자는 [deAPI](https://deapi.ai) 레이어 또는 직접 통합을 통해 이 컴퓨팅 자원에 비용을 지불합니다. 수익은 사용자 보상과 GHX 매입·소각으로 환원되며 — 더 많은 파트너가 합류할수록 이 순환은 자가 강화됩니다.

## 이 문서가 다루는 내용

- **GamerHash** — 우리가 만들고 있는 것, 회사, 비즈니스 모델, 커뮤니티.
- **GamerHash AI** — 앱, 모든 모듈, GHXP 메커니즘, 하드웨어 요구 사항, 수익 모델.
- **deAPI** — GamerHash GPU 네트워크의 공개 개발자 레이어.
- **앱** — 모바일 컴패니언 앱과 GamerHash Store.
- **사용 사례** — 게이머, GPU 기여자, 파트너가 플랫폼으로 할 수 있는 것.
- **로드맵** — 분기별 배포 로그: 출시 완료, 진행 중, 다음 단계.
- **튜토리얼** — 한 번에 끝낼 수 있는 단계별 가이드.
- **리소스** — 유용한 링크, FAQ, 생태계 참고 자료.
- **법적 고지** — 백서 기반 공지, AML/CFT 정책, 면책 조항.
